# Date Night
A web app that finds your location and shows what events are happening that day.

## Languages/Tools Used
- HTML5
- CSS3
- Bootstrap
- JavaScript
- JSON
- AJAX
- API
- JQuery
- Firebase


## Features

- Shows result based on your: 
  - Location 
  - Radius
  - Interests
  - Budget
  
- Shows the pricing of each event

- Features that will be displayed
  - Price
  - Rating
  - Image
  - Peak time
  - Hours
  
## API's used
  - Google Maps
  - Yelp
  - StubHub
  - EventBrite
